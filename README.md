# 🌍 Learn Spanish with Fun

An interactive language learning web application built using HTML, CSS, and JavaScript. The app helps users learn basic Spanish vocabulary through flashcards, quizzes, progress tracking, and fun visual effects.

## 🚀 Features

* 🇬🇧 English to 🇪🇸 Spanish Flashcards
* 📖 Spanish Language Information Section
* 💡 Fun Facts for Every Word
* 📝 Interactive Quiz System
* 🎉 Confetti Celebration on Correct Answers
* 🏆 Level System (Beginner, Intermediate, Expert)
* 📊 Progress Bar
* ⭐ Score Tracking
* 📚 Words Viewed Counter
* 💾 Local Storage Support
* 🎨 Modern Gradient User Interface
* 📱 Responsive Design

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage
* Canvas Confetti Library

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` using Live Server.
4. Start learning Spanish and test your knowledge through quizzes.

## 📌 Project Objective

The objective of this project is to make language learning fun and interactive through flashcards, quizzes, progress tracking, and visual rewards. It helps beginners learn commonly used Spanish vocabulary while keeping them engaged through gamified features.
